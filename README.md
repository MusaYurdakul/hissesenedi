# 📈 Hisse Senedi İstatistiksel Analizi

Python ile hisse senedi verileri üzerinde istatistiksel analiz projesi. CLEBI (Çelebi Hava Servisi) hissesinin 5 yıllık verileri üzerinden analiz yapılmıştır.

## 🔍 Yapılan Analizler

1. **İstatistiksel Momentler** — ortalama, varyans, çarpıklık, basıklık
2. **Korelasyon Analizi** — korelasyon matrisi ve heatmap görselleştirme
3. **Regresyon Analizi** — karşılaştırmalı regresyon modelleri
4. **Hipotez Testleri** — p-değerleri ve karar tabloları

## 📊 Çıktılar

- CSV formatında analiz sonuçları
- PNG grafikler: trend analizi, dağılım, boxplot risk analizi, korelasyon heatmap, hipotez matrisi

## 🚀 Çalıştırma

```bash
python main.py   # Ana analiz
python hipo.py   # Hipotez testleri
```

## 🛠️ Teknolojiler

- Python (pandas, matplotlib, scipy)
